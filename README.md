# ALPHA Init Tool

**ALPHA** is a modern, modular project scaffolding tool designed for rapid stack initialization.

## Features
- **Dynamic Stack Config**: Add/Edit stacks via GUI (`commands.json`).
- **Unified Logic**: Supports Django, FastAPI, Next.js, Vite, and more.
- **Docker Ready**: Auto-generates Dockerfile and docker-compose.yml.
- **Modern UI**: Dark-themed PyQt6 interface with splash screen.

## Installation
1. Clone the repo.
2. Install dependencies: `pip install PyQt6 pyyaml`
3. Run: `python -m alpha` (if installed as package) or integration script.

## Usage
Select your stack, configure options (Docker, UI Framework), and click **Initialize**.
